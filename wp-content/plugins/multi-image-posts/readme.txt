== Description ==

This \`Mutli Image Posts\` plugin will let you create single/bulk post at once with draft status automatically. It is essential plugin for photo blog or there are number of image posts on a wordpress driven site.

**Plugin Main Features are:**

1.  Admin User can only access Plugin's UI.
2.  Start by "Select Image(s)" button for adding (selecting) image to upload from plugin page. If user selects multiple images, multiple box will create for posts.
3.  A box (or multiple ones) create for a post, with image preview display on the left and options on the right.
4.  Form options includes:

*   File name display (minus file extension)
*   Image TITLE, Image ALT TEXT, and Post Title auto-filled with original file name (user can edit)
*   Categories are selectable via available Categories in a blog

**<span style="text-decoration: underline;"><em>upon submitting</em>:</span>**

*   upload Image into Media Library, with TITLE and ALT TEXT being populated properly
*   create Post Type Image
*   Set Uploaded image as "Featured Image". It will automatically insert into the post content
*   Set Post(s) Title
*   Set Image TITLE as "FOCUS KEYWORD" field of the plugin Yoast SEO
*   Set Image TITLE as a Tag for the post(s)
*   Set Image as a Ping for the post(s)
*   Post(s) will automatically be created as DRAFT status
